# Online Shopping Behavior Analysis 🛒📊

Welcome to our first Data Analytics project, developed as part of our academic curriculum. This project focuses on analyzing e-commerce customer behavior to extract actionable business insights using Python.

## 👥 Team Members
* **Besan Ahmad** (Data Preprocessing & Cleaning)
* **Jana Shahrore** (Descriptive & Inferential Statistics)
* **Hala Masri** (Multivariate Correlation Analysis)

## 🚀 Project Pipeline
1. **Data Preprocessing & Cleaning**: Handled missing values (Mean/Mode imputation), rectified logical errors, resolved formatting inconsistencies, and dropped redundant duplicated rows (Final Clean Dataset: 567 records).
2. **Statistical Analysis**: Investigated metrics and demographic distributions.
3. **Correlation Analysis**: Examined multivariate relationships using Correlation Matrices and Heatmaps.
4. **Business Recommendations**: Translated data findings into strategic e-commerce insights (Mobile-First Optimization, Targeted Discounting).

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib

## 📂 Project Files Included
* `Statistics Final Project (final version).ipynb`: The Jupyter Notebook containing full cleaning and analysis source code.
* `online_shopping.csv`: The raw customer behavior dataset (588 rows, 9 features).
* `clean_data.xls`: The final verified and cleaned dataset ready for analysis.
* `Online_Shopping_Behavior_Analysis.docx.pdf`: The comprehensive academic project report.
* `Online-Shopping-Behavior-Analysis (3).pdf`: The presentation slides used during our defense.
